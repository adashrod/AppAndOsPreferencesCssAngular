Strategy 3 (CSS if() w/ container style queries)

This strategy uses a container style query with a custom property and a media query to determine the user's preferred reduced motion setting. It then uses a CSS if() statement to set animation properties based on the preference.

Try changing the app preference in #2 above, and the reduced motion setting in your operating system (accessibility preferences) to see changes.

As of the time of the latest commit, this is supported in Chrome.
