Strategy 3 (CSS if() w/ container style queries w/ custom properties)

This strategy uses a container style query with a custom property and a media query to determine the user's preferred reduced motion setting. It then uses a CSS if() statement to set animation properties based on the preference.

As of the time of the latest commit, it works in Chrome.

Example:

body {
    --anim-duration-long: if(
        style(--app-preference: reduce) or
            (style(--app-preference: system) and media(prefers-reduced-motion: reduce)): 1ms;
        else: 5s;
    );
    --anim-iteration-count-infinite: if(
        style(--app-preference: reduce) or
            (style(--app-preference: system) and media(prefers-reduced-motion: reduce)): 1;
        else: infinite;
    );
}
