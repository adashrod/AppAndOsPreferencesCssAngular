Strategy 3 (CSS vars and classes)

This strategy uses only CSS classes to set the values of animation tokens. Which class to use is calculated in JavaScript; no media queries are used in the CSS.

Example:

body.animate {
    --anim-duration-long: 5s;
    --anim-iteration-count-infinite: infinite;
}

body.reduce {
    --anim-duration-long: 1ms;
    --anim-iteration-count-infinite: 1;
}
