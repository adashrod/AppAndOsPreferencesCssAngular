Strategy 3 (CSS vars and classes)

This strategy uses only CSS classes to set the values of animation tokens.

Example:

body.s3-animate {
    --s3-anim-dur-long: 5s;
    --s3-anim-iteration-count-infinite: infinite;
}

body.s3-reduce {
    --s3-anim-dur-long: 1ms;
    --s3-anim-iteration-count-infinite: 1;
}
