Strategy 2 (container style queries only)

This strategy uses a container style query with a custom property and a media query to determine the user's preferred reduced motion setting.

As of the time of the latest commit, it works in Chrome and Kiwi.

@if (supportsContainerStyleQueriesWithCustomProperties()) {
}

Example:

main {
    --anim-duration-long: 5s;
    --anim-iteration-count-infinite: infinite;
}

@container style(--app-preference: reduce) {
    main {
        --anim-duration-long: 1ms;
        --anim-iteration-count-infinite: 1;
    }
}

@container style(--app-preference: system) {
    @media(prefers-reduced-motion: reduce) {
        main {
            /* rules are duplicated here because of the nested media query */
            --anim-duration-long: 1ms;
            --anim-iteration-count-infinite: 1;
        }
    }
}
