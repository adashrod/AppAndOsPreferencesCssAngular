Strategy 2 (container style queries only)

This strategy uses a container style query with a custom property and a media query to determine the user's preferred reduced motion setting.

Example:

As of the time of the latest commit, it works in Chrome and Kiwi; it doesn't yet work in Firefox (FF doesn't support container style queries w/ custom properties).

main {
    --s2-anim-dur-long: 5s;
    --s2-anim-iteration-count-infinite: infinite;
}

@container style(--ui-rm: reduce) {
    main {
        --s2-anim-dur-long: 1ms;
        --s2-anim-iteration-count-infinite: 1;
    }
}

@container style(--ui-rm: system) {
    @media(prefers-reduced-motion: reduce) {
        main {
            /* rules are duplicated here because of the nested media query */
            --s2-anim-dur-long: 1ms;
            --s2-anim-iteration-count-infinite: 1;
        }
    }
}
